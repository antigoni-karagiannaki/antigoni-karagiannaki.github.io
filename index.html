What I Do

Data Analysis

I have a strong background in Data Analysis , utilizing Python to process and analyze data, including calculating coefficients and extracting insights from structured and unstructured datasets. During my internships, I worked extensively with numerical analysis, statistical modeling, and neural data evaluation. Additionally, I have completed coursework in Information Retrieval Systems, covering key topics such as Boolean, Vector Space, and Probabilistic Retrieval Models, query expansion techniques, document clustering, and text indexing. My expertise extends to search algorithms, multimedia retrieval, and web-scale information systems, providing me with a deep understanding of how data is stored, processed, and retrieved efficiently.

Web Design

I have experience in full-stack web development, capable of building fully functional and responsive websites using HTML, CSS, and JavaScript. I am also proficient in integrating PHP and MySQL for database management, ensuring dynamic and data-driven functionality. My skill set includes frontend UI/UX design, optimizing user experience with interactive elements and mobile-friendly layouts. With a keen eye for detail and performance optimization, I develop structured, scalable, and efficient web applications tailored to various needs.

Machine Learning

My work in Machine Learning (ML) and Deep Learning has focused on Active Learning strategies and query selection techniques, which I explored in depth during my thesis. I have implemented and evaluated models using supervised, unsupervised, and reinforcement learning techniques, optimizing data efficiency in training processes. I am proficient in designing and applying uncertainty-based, diversity-based, and hybrid query selection strategies, ensuring that machine learning models improve their performance with minimal labeled data. My expertise also includes working with deep neural networks, model fine-tuning, and large-scale dataset handling, contributing to the development of more efficient and adaptive AI systems
